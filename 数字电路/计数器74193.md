
| **引脚符号**              | **名称**  | **功能描述**                                  |
| --------------------- | ------- | ----------------------------------------- |
| **$CP_U$**            | 加计数时钟输入 | 上升沿触发，计数加 1（此时 $CP_D$ 应保持高电平）             |
| **$CP_D$**            | 减计数时钟输入 | 上升沿触发，计数减 1（此时 $CP_U$ 应保持高电平）             |
| **$MR$**              | 异步清零端   | **高电平有效**。只要为 1，输出 $Q_3 \sim Q_0$ 立即全为 0  |
| **$\overline{PL}$**   | 异步并行置数端 | **低电平有效**。为 0 时，将 $P_3 \sim P_0$ 的数据送入输出端 |
| **$P_0 \sim P_3$**    | 并行数据输入端 | 预置数的数值来源                                  |
| **$Q_0 \sim Q_3$**    | 计数输出端   | $Q_0$ 为最低位 LSB，$Q_3$ 为最高位 MSB             |
| **$\overline{TC}_U$** | 进位输出端   | 加计数溢出（从 15 到 0）时产生低电平脉冲                   |
| **$\overline{TC}_D$** | 借位输出端   | 减计数溢出（从 0 到 15）时产生低电平脉冲                   |
**$CP_U$这个是用于加的时钟序号**
PL是用于置数的
MR是用来置零的
