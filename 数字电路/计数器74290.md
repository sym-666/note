| **引脚符号**                 | **名称**     | **功能描述**                  |
| ------------------------ | ---------- | ------------------------- |
| **$CP_A$**               | 时钟 A 输入    | 二进制计数器时钟（下降沿触发）           |
| **$CP_B$**               | 时钟 B 输入    | 五进制计数器时钟（下降沿触发）           |
| **$Q_A$**                | 输出 A       | 二进制计数器输出（最低位 LSB）         |
| **$Q_B, Q_C, Q_D$**      | 输出 B, C, D | 五进制计数器输出（$Q_D$ 为最高位 MSB）  |
| **$R_{0(1)}, R_{0(2)}$** | 直接清零端      | 两端同时为高时，输出全部清 0           |
| **$S_{9(1)}, S_{9(2)}$** | 直接置 9 端    | 两端同时为高时，输出置为 1001 (BCD 9) |
| **$V_{CC}, GND$**        | 电源与地       | 通常为 +5V                   |
是由一个二进制以及一个5 进制的计数器组成的，可以来构成2到10进制计数器 

```ad-note
title:注意事项
对于中规模组合电路而言， 0 和A 一般代表低位，而4 和 D一般代表的是高位。
```

对于两个计数器而言可以把它们串在一起，可以达成计更大的数的目的。比如串联模6和模10 计数器可以实现计数60.
